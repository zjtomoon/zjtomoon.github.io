Ubuntu编译安装emacs

Jul 3, 2021  

安装必要的依赖

sudo apt-get install build-essential

安装emacs需要的依赖

sudo apt-get build-dep emacs24

解压emacs源码包

tar -zxvf emacs-26.1.tar.gz -C ~/softwares

编译emacs

cd ~/software/emacs-26.1/
./config
make
##(optionally) install emacs to /usr/local/bin
sudo make install
