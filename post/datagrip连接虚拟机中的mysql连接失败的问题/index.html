Datagrip连接虚拟机中的mysql连接失败的问题

Jul 3, 2021  

创建对外网开放的mysql用户

create user 'tom' @ '%' identified by '123';
grant all privileges on *.* to 'tom'@'%' identified by '123';
FLUSH PRIVILEGES;

修改my.cnf

vim /etc/mysql/my.cnf

##注释bind-address
#bind-address		127.0.0.1
bind-address		0.0.0.0

重启mysql

sudo /etc/init.d/mysql restart
