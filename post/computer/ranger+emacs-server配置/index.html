Ranger+emacs Server配置

Jul 10, 2021  

安装pypi版ranger

sudo pip3 install ranger-fm

开启emacs服务

emacs --daemon
# 开启emacs服务，把emacs配置文件加载到内存中

设置emacs为默认编辑器

vim ~/.bashrc
##添加一行
export EDITOR=emacsclient
source ~/.bashrc
